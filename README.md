# 🛒 E-Commerce React Website

A modern and responsive E-Commerce web application built using **React.js**. This project fetches product data from an API and provides full shopping functionality including product listing, cart management, and more.

---

## 🚀 Features

* 🛍️ Product Listing (API-based)
* 🔍 Product Details Page
* ➕ Add to Cart Functionality
* 🧮 Dynamic Cart (Quantity & Total Price Calculation)
* ❌ Remove Items from Cart
* 📄 About Page
* 🏠 Home Page
* ⚡ Fast and Responsive UI
* 💰 Price Formatting (INR)

---

## 🧑‍💻 Tech Stack

* **Frontend:** React.js
* **State Management:** Context API
* **Styling:** CSS / Styled Components
* **API:** External Product API
* **Routing:** React Router


---

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/Aditya-Chauhan-Dev/E-commerce-Website.git
```

2. Navigate to the project folder:

```
cd E-commerce-Website
```

3. Install dependencies:

```
npm install
```

4. Start development server:

```
npm run dev
```

---

## 🌐 API Integration

* Products data is fetched from an external API
* Dynamic rendering of products on the Products page

Example:

```
fetch("https://dummyjson.com/products")
```

---

## 🛒 Cart Functionality

* Add products to cart
* Increase/Decrease quantity
* Remove items
* Calculate subtotal automatically

---

## 💡 Key Components

* **AddToCartBtn** → Handles adding items to cart
* **CartContext** → Manages global cart state
* **FormatPrice** → Converts and formats price into INR
